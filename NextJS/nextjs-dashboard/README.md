## Next JS Hello World

A hello world project for nextjs learning.

## how to run

1. fill up the .env file with all the values
1. npm install
1. npm run dev

## note about database

1. during testing, I used the neon db from the vercel dashboard. 
1. run the seed using the seed URL, http://localhost:3000/seed
