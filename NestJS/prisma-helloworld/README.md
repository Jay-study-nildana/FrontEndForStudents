# Prisma Hello World

Prisma features like schema, setting up the prisma client, migrations as well as seeding.

# .env file

a .env.example has been provided. fill up the neccessary details. 

note: this does not show usage of prisma client in your controller. It only shows db relations action of prisma inside a nest project

# how to run

```

npm install
npx prisma generate (creates the prisma client)
npx prisma migrate deploy (deploy the existing migrations)
npm run prisma:seed 
npx prisma studio (see the tables in the local web UI)

```

# some additional notes about prisma

```
Commands to use after adding a more complex schema

npx prisma validate --schema=prisma/schema.prisma (before deploying validate)
npx prisma format --schema=prisma/schema.prisma (simple validations are fixed)
npx prisma migrate dev --name complextables (generate and apply migrations)
npx prisma generate (generate new client as per the new updated schema)
```
