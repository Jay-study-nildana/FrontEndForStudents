# Front End for NestJS Five

A simple react js project with tailwind configured. Built around F1 theme. 

Illustrates the following concepts. 

1. Standard React JS features like Tailwind integration, Routing, Navbar and Footer and so on. 
1. Loading data from a local data file, "lapData.ts". 
1. Loading data from that local data file, via a service layer, simulating a "network service".
1. Loading animation for individual data items.
1. Summary stat generation and display using existing data services.
1. UI elements such as tables, tab navigation within a page, horizontal scrolling.
1. Also, a 'local' SVG image generator, that fills all the images.
1. An SVG based 'animation' which displays an interactive track racing.
1. Multiple "Page Not Found" components with several variations.

Also, 

1. Integrates with the NestJS BackEnd
1. Login, Logout and Register
1. CRUD operation views
1. Filed Upload and display
1. Content display from blog themed endpoints
1. Search Input and Display with Pagination

## env file

create .env file using the example env file and fill all details.

## commands

```
  npm install
  npm run dev
```
