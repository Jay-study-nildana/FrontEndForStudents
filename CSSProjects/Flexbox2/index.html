Flexbox (Flexible Box Layout)

  • Use Flexbox for 1-dimensional layouts (row OR column)
  • Main concepts: main axis (flex-direction) and cross axis
  • Container properties:
    • display: flex; flex-direction; flex-wrap
    • justify-content (align along main axis)
    • align-items (align along cross axis)
    • align-content (align multi-line content when wrapped)
    • gap (space between items)
  • Item properties:
    • flex: <grow> <shrink> <basis>
    • order (change visual order)
    • align-self (override align-items per item)
  • Common patterns: horizontal nav, equal-height cards, centered content, responsive rows that wrap
