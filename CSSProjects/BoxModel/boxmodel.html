Box Model — Demo

Explore content-box vs border-box, padding, border, margin, and margin collapse.

1. box-sizing: content-box (default)

Width = content width. Padding & border are added outside the width.

content-box
width:200px; padding:20px; border:6px

2. box-sizing: border-box

Width includes padding & border — easier to reason about sizing.

border-box
width:200px; padding:20px; border:6px

3. Margin collapse

Adjacent vertical margins can collapse — top/bottom margins between block elements.

Element A (margin-bottom: 30px)
Element B (margin-top: 50px)

The vertical gap is the larger margin (50px), not 80px — that's margin collapse.

4. Practical rule (recommended)

Apply a global box-sizing rule to make layout predictable:

*, *::before, *::after { box-sizing: border-box; }

Try resizing the window. Use devtools to inspect computed box sizes.
