# Full Stack Notes Project

This is what this project, front end and backend combo, demonstrate.

1. Prisma + Postgres CRUD controller demonstrating Create/Read/Update/Delete.
1. Auth endpoints: register and login that issue JWTs containing role claims.
1. Role-based access enforced in Express via middleware (authenticate + requireRole).
1. Route-level gating (router.use, per-route middleware) with controller functions focused on business logic.
