# React JS + PWA + Notifications

React JS project that does PWA plus notifications. 

note: the backend server project needs to be running for all features to work. 

## commands

```
  npm install
  npm run dev
  
```

# things to do PWA

1. Set your browser to offline to test offline capabilities (under developer tools)
1. send notification and recieve it
1. open multiple browsers and access this web app. send and recieve notifications on all of them.
1. notifications can be delayed. sometimes it is instantaneous and other times, notifications can come after many minutes.
