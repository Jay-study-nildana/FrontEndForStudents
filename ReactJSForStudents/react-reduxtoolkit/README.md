# React JS + Redux Tool Kit

React project that show cases usage of redux tool kit. 

## NASA APOD

The project uses the NASA APOD 3rd party API to illustrate consuming a real web api. Please update the 'VITE_NASA_API_KEY' in the .env file if you want to see that feature in action. 

API key can be obatined, here, https://apod.nasa.gov/apod/

note : project will still work without the APOD section. 

## commands

```
  npm install
  npm run dev
```
